## Installation 

1. Duplicate `.env.example` and rename to `.env`. Modify value based on preference.
2. Install composer:
    ```
    composer install
    ```
3. Generate application key:
   ```
   php artisan key:generate
   ```
4. Run migration:
   ```
   php artisan migrate
   ```
5. Run seeder to generate random data:
   ```
   php artisan db:seed
   ```
